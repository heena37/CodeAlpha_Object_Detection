# 🎯 Object Detection App

This project is an AI based Object Detection Application using YOLO, OpenCV, Python and Streamlit.  
Application will enable users to upload an image and automatically identify objects in an image using a pre-trained AI model.

## Features:

-Upload images for detection of objects.
-Automatically detect objects in the real world
-Show objects found with bounding boxes.
-Object recognition using YOLO (AI technology)
-Simple and easy to use interface

## Technologies Used:

- Python
- Streamlit
- OpenCV
- YOLO (Ultralytics)
- NumPy
